    • ANALYSE DATASET

    DATASET 1, SEGURIDAD DE LA ZONA

    Tipo Descripción
    PROVINCIA string Contiene el nombre de la provincia
    SEXO string Contiene el género
    AÑO float Contiene la proporción de personas que se sienten seguras en su zona en el año indicado
    RANGO(valor de los datos) float 58-87

    El dataset se mide mediante porcentajes.

    DATASET 2, DOTACIÓN POLICIAL

    Tipo Descripción
    CÓDIGO DE MUNICIPIO float Contiene el número del municipio
    MUNICIPIO string Contiene el nombre del municipio
    AÑO float Contiene la dotación policial de la zona en el año indicado
    RANGO(valor de los datos) float 0-3.96

    El dataset se mide mediante porcentajes.

    DATASET 3, ÍNDICE DE DELITOS

    Tipo Descripción
    CÓDIGO DE MUNICIPIO float Contiene el número del municipio
    MUNICIPIO string Contiene el nombre del municipio
    AÑO float Contiene el índice de criminalidad de la zona en el año indicado
    RANGO(valor de los datos) float 0-170

    El dataset se mide mediante una tasa por mil habitantes.

    DATASET 4, INFRACCIONES PENALES

    Tipo Descripción
    PROVINCIA string Contiene el nombre de la provincia
    AÑO float Contiene el índice de infracciones penales de la zona en el año indicado
    RANGO(valor de los datos) float 30.32-38.9

    El dataset se mide mediante una tasa por mil habitantes.

    RESOURCE NAMING STRATEGY

    DOMINIO

    https://www.vascomap.com/

    Las uris se dividen en dos tipos:

  • Uris para definir la ontología
  • Uris para definir los recursos inicializados

    • URIS DE LA ONTOLOGÍA

    El dominio elegido para nuestras URIs propias es https://www.vascomap.com/, el cual hemos verificado que no está en uso.

    Todas las ontologías(propias) comienzan por: https://www.vascomap.com/ontology#

    A su vez, este tipo de uris pueden dividirse en:

  • Uris para definir las clases:
    • Observation: definimos 4 subclases (exhaustive disjoint)
      • DotPol
      • IndDel
      • InfPen
      • SegZon
    • Municipality
    • Province
  • Uris para definir propiedades de objeto:
    • takesPlaceInProv
    • takesPlaceInMun
    • has
    • belongsTo
  • Uris para definir propiedades de datatype:
    • name
    • id
    • code
    • value
